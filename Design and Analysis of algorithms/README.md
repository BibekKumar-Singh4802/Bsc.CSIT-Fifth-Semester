# 📘 Design and Analysis of Algorithms Lab

## 📂 Project Overview

This repository contains all lab programs for the **Design and Analysis of Algorithms (CSC314)** course.  
It covers fundamental to advanced algorithms with:

- Time Complexity Analysis (RAM Model)
- Execution Time Measurement (`clock()`)
- Standard Algorithm Implementations in C
