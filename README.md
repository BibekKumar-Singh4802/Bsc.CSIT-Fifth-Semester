## Cryptography & Security Lab

This repository contains **Cryptography and Security Lab programs** for **BSc CSIT Fifth Semester**,  
designed according to the **Tribhuvan University (TU)** syllabus.  
The programs focus on classical and modern cryptographic algorithms, authentication protocols,  
and basic Linux/network security configurations.
